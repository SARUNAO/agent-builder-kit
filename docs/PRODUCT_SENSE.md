# Product Sense

## このプロジェクトは何か
- agent-builder の実運用を題材に、mdBook でワークショップ記事サイトを構築するプロジェクト

## 人間要求
- 何もない新規リポジトリから、agent-builder を使って mdBook のワークショップ記事サイトを作りたい。ワークショップ自体で plan-manager -> task-planner -> task-worker の流れも体験できるようにする。

## 明確化された目的
1. agent-builder-kit の実運用テストと、その記録を残す
2. みんなに使ってもらうためのチュートリアルを作る
3. mdBook を、AI と docs 駆動フローで作る題材として使う

## 価値仮説
- 実際の開発対象を mdBook にすることで、AI による docs 駆動開発フローを end-to-end で見せやすい
- Zenn や Note より構成管理の自由度が高く、ワークショップ教材として再利用しやすい
- 単なる記事執筆ではなく、GitHub 公開まで含めた開発運用の練習台になる

## 成功イメージ
- GitHub 上で `agent-builder-kit` と関連する mdBook チュートリアルが公開されている
- 参加者が local で追従しつつ、最終的に公開物も参照できる
- 実運用してみて良かった点と詰まった点が、ワークショップ本文か補助 docs に残る
