# Decision Log

この docs は、実装中に確定した判断と、その時点の理由だけを短く残す一次記録の置き場。

## 書き方
- 1 件 1 行から 3 行で残す
- 何を決めたか、なぜそうしたかを先に書く
- 後から本文に移す前提なので、感想より事実を優先する
