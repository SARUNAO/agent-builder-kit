# Design

## 前提
- project mode: `new`
- profile: `standard`
- role model: `four_role_default`
- packs: obsidian_canvas_pack

## 生成方針
- docs を source of truth とする
- role skill によって plan / chunk / ticket を更新する
- 必要なら `.canvas` を派生物として再同期する

## 初期 seed
- 初期 block として `何を作るか決める` を置く
- `プランオーナー` が聞き取りを通じて block を増やしていく
